# Annotations Dashboard
## A set of tools and applications to monitor the development and status of the training dataset

**[Global Training Dataset Statistics]([https://example.com](https://lookerstudio.google.com/reporting/b6fd8fb9-9b81-45d8-a99f-36b88dcd80e1/page/MIJeF))**  
An application for visualizing summary statistics on the current state of the global training dataset, offering multiple views across several parameters with filtering options.
